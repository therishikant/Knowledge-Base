# ? Markdown Notes (Quick Reference)

Markdown is a lightweight markup language used to format text.

---

## 1. Headings

Use `#` for headings:

# H1
## H2
### H3
#### H4

---

## 2. Text Formatting

- Bold ? **text** ? `**text**`
- Italic ? *text* ? `*text*`
- Strikethrough ? ~~text~~ ? `~~text~~`
- Inline Code ? `code` ? `` `code` ``

---

## 3. Lists

### Unordered List
- Item 1
- Item 2
    - Subitem

### Ordered List
1. First
2. Second
3. Third

---

## 4. Links

[Google](https://google.com)

Syntax:
`[Text](URL)`

---

## 5. Images

![Alt Text](image.png)

Syntax:
`![Alt](image-url)`

---

## 6. Code Blocks

Use triple backticks:

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello");
    }
}
```
