# VS-Code Setup

## Extensions
- Markdown All in One → shortcuts, TOC, formatting
- Markdown Preview Enhanced → better preview
- Foam (optional but powerful) → turns VS Code into Obsidian-like system
  - Link [https://foamnotes.com/]
- Excalidraw -> Creating diagram 

## Shortcuts
### Markdown Preview Enhanced
- Ctrl + Shift + V
- Ctrl + K, V

### Foam
- Link topics using [[links]] VS Code + Foam
